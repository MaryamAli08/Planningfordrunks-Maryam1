Introducing the Planning for Drunks project

This project was developed as part of an assignment for geography programming course. There were several projects to choose from however, I choose to conduct the planning for drunk project. The project consists of exploring a town full of drunks and creating a program which guides drunks home. Futhermore, this website will explore each step in producing the drunks model. For more information , please press learn more. This will also guide you to the github page which holds the model informations.

About the development of the Project

The Planning for drunks project was introduced to produce an algorithm which examine a town full of drunks and help the drunks walk home from the pub. This project will be produced in Python, Sypder. Python is high level programming lanaguage which is successful at creating algorithms , softwares and creating data visualisations.

Futhermore, the program will be implemented using a data file which contains the pub (the drunks are leaving ) and their homes (10, 250).The program will need to assign the drunks to a home before leaving the pub, allow the drunks to wonder around the town until they reach their assigned home. Additionally, the program also aims to exemplify the density of drunks passing through each point on a map.

How to build the Drunks Program

1. Introduce the data file to the model by employing the function import 'csv'.

2. Exemplify the pub points and home points by using built-in functions such as 'enumerate'

3. Illustrate the drunks leaving the pub and their route home.

4. Examine the density of the drunks passing through each point of the map.

5. The final step of project is to save the density map as a text file.

Data Visualisations

To create a visualisation of the drunks route home, the import matplotlib.pyplot function was employed. The matplotlib.pyplot is an object oriented plotting library which enables to figures and plots to be created.

This projects will exemplify using the matplotlib.pyplot , the drunks walk home and also will highlight when the drunks arrive at their assigned house. Also, the matplotlib.pyplot will be adopted to illustrate the density.

Software Issues

There were several issues which occurred whilst building the drunks model. One issue was all the drunks were unable to return home, and continued to wonder in the middle of the town. This was a concern as the aim of the project to build an algorithm which walks drunks to their assigned homes.

This was successfully resolved by increasing the numofiterations by a few hundred. This helped to increase the chances of the drunks locating their house. Though, the increase in the num of iterations did result in a slower execution time, therefore the drunks do take longer to arrive home. Overall, the drunks did successfully arrive homesweethome.
